Blackjack game

This project is a simple Python card game where you play against the computer. It gives you random cards and calculates your score while letting you choose whether to draw more cards or stop. If your total goes over 25, you lose. After your turn, the computer reveals its cards, compares scores, and decides the winner.

Caesar cipher

This project is a Python program that works as a Caesar cipher tool. It takes a message and a shift number from the user, then shifts each letter forward or backward in the alphabet to encrypt or decrypt the text. It works by finding each letter’s position, changing it using the shift value, and printing the new message.

Bidding auction

This project is a Python bidding system that finds the highest bidder. It takes names and bid amounts from multiple users and stores them in a dictionary. The program keeps asking for new bidders until no one is left. Then it checks all bids, compares them, and prints the winner with the highest bid.

Followers guessing game

This project is a simple Python game where players guess which famous person has more Instagram followers. It shows two personalities with their details, lets the user choose A or B, and keeps track of the score until a wrong answer ends the game.

Number guessing game

This project is a number guessing game where the computer selects a random number between 1 and 100 and the player has to guess it within limited attempts based on the chosen difficulty. The game gives hints like too high or too low after each guess and ends when the player wins or runs out of chances.
